Change Logo

Its very easy to change your logo.
All you have to do is open your desired html file on a code editor and change the source/name of the logo to your logo.

The white logo only shows in case you are using a transparent header, if you don't intend to use one you might just skip this step.

Any-file.html 
                                            
<a class="navbar-brand" href="index.html">
    <!-- INSERT YOUR LOGO HERE -->
    <img src="assets/images/logo.svg" alt="logo" width="150" class="logo">
    <!-- INSERT YOUR WHITE LOGO HERE -->
    <img src="assets/images/logo-white.svg" alt="white logo" width="150" class="logo-white">
</a>

Boxed Layout

To change the layout of the Template from wide to boxed all you have to do is insert the class "boxed" on the "wrapper" div of the HTML page that you want to change the layout, like the example below:

index-02.html 
                                            
<!-- BEFORE - Wide Layout -->
<div class="wrapper">
.....


<!-- AFTER - Boxed Layout -->
<div class="wrapper boxed">
.....

Header Options

If you want to make your header sticky insert the class "sticky" inside your header like in the example below:

any-file.html 
                                            
<!-- =============== START OF HEADER NAVIGATION =============== -->
<!-- Insert the class "sticky" in the header if you want a sticky header -->
<header class="header sticky">
....
....
</header>
<!-- =============== END OF HEADER NAVIGATION =============== -->

If you want to have a fixed and transparent header you will have to add the classes "header-fixed header-transparent" and if you want your header to have white text add the class "text-white" too in your header like in the example below:

index.html 
                                            
<!-- =============== START OF HEADER NAVIGATION =============== -->
<!-- Insert the class "sticky" in the header if you want a sticky header -->
<header class="header header-fixed header-transparent text-white">
....
....
</header>
<!-- =============== END OF HEADER NAVIGATION =============== -->

Page Header Background

To change the background of your page header all you have to do is change the image that can be found in the inline css on the page header div like in the example below:

any-file.html 
                                            
<!-- =============== START OF PAGE HEADER =============== -->
<!-- Change the url to the image you want to have as a background -->
<section class="page-header overlay-gradient" style="background: url(assets/images/image-name);">
....
....
</section>
<!-- =============== END OF PAGE HEADER =============== -->

Contact Form

Contact form will use php to send emails. So open email.php file that can be found in the assets/php folder. Change the current email "your-email@movify.com" to your email address and you are ready to go.


NOTE: Contact form require web server to send emails. Emails can go to your spam/junk folder.

assets/php/email.php 
                                            
########################################################################
########################################################################

$site_title     = "Movify"; //Site Title
$to_email       = "your-email@movify.com"; // <====== Replace with your email address
$subject        = "Movify - Email "; //Replace with your email default subject

Subscription

To connect your mail chimp list, open custom.js that can be found in the assets/js folder. Find the following script and place your mailchimp list url into the "url" as shown on the example below below.

assets/js/custom.js 
                                            
/*----------------------------------------------------
  MAILCHIMP
----------------------------------------------------*/
$('.mailchimp').ajaxChimp({
    callback: mailchimpFunct,
    url: "your-mailchimp-url-here" // <===== Replace this with your own mailchimp post URL. Paste the url inside "".  
});
